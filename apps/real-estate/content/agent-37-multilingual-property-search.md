## 2. **Machine Translation for Property Listings Across Languages**

### Domain-Specific Translation Challenges:
- **Real estate terminology** (e.g., "condo", "townhouse", "freehold")
- **Local measurement units** (square meters vs. square feet)
- **Cultural concepts** (e.g., "garden" vs. "yard")
- **Legal terms** (tenure types, ownership structures)
- **Local amenities** (school districts, transportation terms)

### Technical Solutions:
- **Domain-adapted MT models** (RealEstateMT)
- **Terminology-aware translation** with real estate glossaries
- **Context-aware translation** considering property type and location
- **Post-editing MT** with real estate expert feedback
- **Multimodal translation** (text + images + floor plans)

### Quality Evaluation Metrics:
- **BLEU scores** for translation quality
- **Domain-specific accuracy** for real estate terms
- **Cultural appropriateness** scores
- **Legal compliance** verification
- **User satisfaction** in property discovery tasks

## 3. **Multilingual Embeddings for Property Similarity**

### Embedding Approaches:
- **Multilingual BERT** (mBERT) for property descriptions
- **XLM-R** (Cross-lingual Language Model RoBERTa)
- **LASER** (Language-Agnostic SEntence Representations)
- **Real estate domain-adapted multilingual embeddings**
- **Cross-modal embeddings** (text + images + location)

### Similarity Metrics:
- **Cross-lingual semantic similarity** for property descriptions
- **Feature-based similarity** across languages
- **Price-normalized similarity** considering local markets
- **Cultural preference-weighted similarity**
- **Temporal similarity** for market conditions

### Applications:
- **Cross-lingual property recommendation** systems
- **International market comparison** tools
- **Price prediction** across different language markets
- **Investment opportunity discovery** in foreign markets

## 4. **Cultural Adaptation in Property Description Translation**

### Cultural Factors in Real Estate:
- **Living preferences** (open plan vs. separate rooms)
- **Privacy norms** (garden walls, window placements)
- **Status symbols** (specific amenities as status markers)
- **Family structure** influences on property needs
- **Climate adaptation** features

### Adaptation Techniques:
- **Cultural annotation** of property descriptions
- **Adaptive translation** based on target culture
- **Cultural preference modeling** from user behavior
- **Local market knowledge** integration
- **Expert validation** of cultural adaptations

### Research Questions:
- How do property description conventions vary across cultures?
- What cultural factors most influence property preferences?
- How to automatically detect cultural mismatches in translations?
- What adaptation strategies work best for different property types?

### Integration Approaches:

- **Federated search** architectures

### Business/Regulatory Challenges:
