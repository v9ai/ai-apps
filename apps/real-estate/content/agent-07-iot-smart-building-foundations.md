## **5. Communication Protocols and Data Standards**

### **Protocol Landscape:**

| Protocol | Primary Use | Key Features | IoT Integration |
|----------|-------------|--------------|-----------------|
| **BACnet** | Building automation | Object-oriented, standardized | BACnet/WS, BACnet/SC |
| **Modbus** | Industrial control | Simple, widely adopted | Modbus TCP, Modbus RTU |
| **MQTT** | IoT messaging | Lightweight, pub/sub | MQTT-SN, Sparkplug B |
| **Haystack** | Semantic modeling | Tag-based metadata | Brick schema integration |
| **OPC UA** | Industrial IoT | Information modeling, security | Companion specifications |

### **Emerging Standards:**
- **Project Haystack 4.0** with enhanced semantic capabilities
- **Brick Schema** for building metadata standardization
- **SAREF4BLDG** (Smart Applications REFerence ontology)
- **IFC (Industry Foundation Classes)** for BIM integration
