# RL-Based Crawlers Evaluation Framework

**Authors.** Anonymous

**Venue / year.** 2025

**Links.** [DOI](https://doi.org/10.5281/zenodo.15077395) · [source](https://zenodo.org/records/15077395)

**Abstract.**

This artifact contains the framework for implementing Reinforcement-Learning(RL)-based crawlers and the implementations of the Multi-Armed Krawler (MAK) proposed in the paper titled _Less is More: Boosting Coverage of Web Crawling through Adversarial Multi-Armed Bandit_ accepted at the 55th Annual IEEE/IFIP International Conference on Dependable Systems and Networks (DSN 2025). This repository also contains the code for implementing the baselines WebExplor and QExplore, the used PHP and Node.JS-based web applications.
