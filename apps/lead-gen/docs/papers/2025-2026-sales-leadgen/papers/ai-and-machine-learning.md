# AI and Machine Learning

**Authors.** Neya Global, Anna Neya Kazanskaia

**Venue / year.** 2025

**Links.** [DOI](https://doi.org/10.64357/ai-and-machine-learning-2025) · [source](https://doi.org/10.64357/ai-and-machine-learning-2025)

**Abstract.**

AI and machine learning are transforming the way non-profits operate, offering tools for data analysis, automation, and personalized engagement. This book introduces key concepts, applications, and ethical considerations for adopting AI in mission-driven work. It provides practical examples and strategies to ensure responsible and impactful use of these technologies.
