# Customer Churn Prediction Using Machine Learning Models

**Authors.** Kripa Singh

**Venue / year.** 2026

**Links.** [DOI](https://doi.org/10.5281/zenodo.19022654) · [source](https://zenodo.org/records/19022654)

**Abstract.**

ABSTRACT&nbsp;
Customer churn prediction is a critical task for organizations seeking to maintain long-term profitability and competitive advantage. Customer churn occurs when customers discontinue using a company&rsquo;s services or products. Early identification of customers who are likely to churn allows organizations to implement targeted retention strategies. Traditional statistical approaches often struggle to capture complex patterns in customer behavior. Machine learning models provide powerful tools for analyzing large volumes of customer data and identifying hidden patterns related to churn behavior. This study proposes a machine learning-based framework for predicting customer churn using algorithms such as Logistic Regression, Decision Tree, Random Forest, and Gradient Boosting. The dataset includes customer demographic information, service usage behavior, billing details, and support interactions. Model performance is evaluated using Accuracy, Precision, Recall, and F1-score metrics. Experimental results indicate that ensemble learning models achieve higher predictive accuracy compared to traditional models and can effectively support customer retention strategies.
Key words: Customer Churn Prediction, Machine Learning, Predictive Analytics, Customer Retention, Data Mining
