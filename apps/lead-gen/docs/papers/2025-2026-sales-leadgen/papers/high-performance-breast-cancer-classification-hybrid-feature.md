# High-Performance Breast Cancer Classification Hybrid Feature Selection and Gradient-Boosted Ensemble Learning

**Authors.** Pinakshi Panda, S. Bisoy, Abhimanyu Patra, Abhishek Das, Prabodh Kumar Sahoo, Debendra Kumar Sahoo

**Venue / year.** 2025 International Conference on Cognitive, Green and Ubiquitous Computing (IC-CGU) · 2025

**Links.** [source](https://www.semanticscholar.org/paper/c63b6f48750dab5838d72d4c794455e5817c65f0)

**Abstract.**

The correct diagnosis and prognosis of cancers is one of the key issues of bioinformatics especially with the application of high-dimensional microarray gene expression data. This paper introduces a new machine learning model to identify breast cancer, which combines a hybrid Whale Optimization Algorithm Sparrow Search Algorithm (WOA–SSA) of feature selection with an Extreme Gradient Boosting (XGBoost) classifier. The evaluation was done using a Breast Cancer microarray dataset of 181 samples with 12,533 features of gene-expressions. The hybrid, WOA-SSA algorithm was quite helpful in dimensionality reduction by finding a small, biologically relevant set of genes, using the global exploration of WOA with the high local exploitation shown by SSA. These features were then optimized and then classified using XGBoost, which guaranteed high predictive stability using gradient-boosted ensemble learning. Accuracy, precision, recall, specificity, and F1-score were the strictly used parameters to measure performance. The classification accuracy of the proposed framework was 95% which is better than the traditional feature selection and classification methods. The findings highlight potential of integrating more sophisticated metaheuristic optimization and potent ensemble classifiers to improve breast cancer diagnosis, hence, lead to precision oncology and personalized care.
