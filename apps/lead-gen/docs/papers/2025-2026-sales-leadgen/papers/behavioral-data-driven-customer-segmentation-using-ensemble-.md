# Behavioral Data–Driven Customer Segmentation using Ensemble Machine Learning and Temporal Forecasting

**Authors.** Chettiyar Vani Vivekanand, A. A

**Venue / year.** 2025 4th International Conference on Automation, Computing and Renewable Systems (ICACRS) · 2025

**Links.** [source](https://www.semanticscholar.org/paper/d94266a9bc2f1693fe727aa61a026cf0f65a7160)

**Abstract.**

Customer segmentation is the basis to improve the effectiveness of customer relationship management (CRM) and targeted marketing. The document presents a different ensemble-based classification model which is about the prediction power of gradient boosting and deep learning methods to better categorize customers into different behavioral segments. The new model combines the Light Gradient Boosting Machine (LightGBM) which is a lightweight model that can handle categorical features, into a Keras-based deep neural network that has the ability to connect complex non-linear interactions. Label encoding, feature standardization, and one-hot encoding are the methods that have been used to make the features compatible across different learners in the dataset processing phase.The base model's output is fed to the meta-learning classifier, which is the one that considers these outputs and decides on a final prediction through the stacked ensemble. The design not only brings about higher accuracy of the classification but also stability of the models and interpretability. On the real-world customer data, the model is tested with cross-validation and standard classification metrics and outperforms several normal standalone models. The results can be seen as an endorsement of the hybrid learning application in getting the customer deep insights, thus facilitating an intelligent and scalable approach to dynamic segmentation, churn forecasting, and advertising.
