# Version v2.0.0 — Reinforced-lib: Rapid prototyping of reinforcement learning solutions

**Authors.** Wojnar, Maksymilian

**Venue / year.** 2025

**Links.** [DOI](https://doi.org/10.5281/zenodo.17380335) · [source](https://zenodo.org/records/17380335)

**Abstract.**

Reinforced-lib is a Python library for rapid development of reinforcement learning (RL) solutions, designed with user-friendliness and extensibility in mind. Built on JAX, it supports both deep RL and traditional agents, providing a rich ecosystem and comprehensive documentation. This release introduces several major enhancements: support for vectorized environments, implementation of proximal policy optimization for discrete and continuous action spaces, and integration of neuroevolution algorithms as an alternative to RL approaches. Additional improvements include a masked multi-armed bandit (MAB) agent, new schedulers with an MAB interface, documentation updates, and bug fixes.
This research was funded by the National Science Centre, Poland (DEC-2023/05/Y/ST7/00004).
