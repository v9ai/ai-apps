# Adaptive Hybrid Agents for CRM: Combining Rule Engines, LLMs, and Feedback Loops for Reliable Automation

**Authors.** Nikita Kothari, Vinay Venkatesh, Varun Joshi, Lav Kumar

**Venue / year.** 2026 9th International Conference on Electronics, Materials Engineering & Nano-Technology (IEMENTech) · 2026

**Links.** [source](https://www.semanticscholar.org/paper/6cbcd5b028f1c7eff04bf73298b45006bbe2211e)

**Abstract.**

As enterprise systems evolve from deterministic automation toward adaptive agentic intelligence, the challenge of balancing creativity, compliance, and reliability has become central to Customer Relationship Management (CRM) automation. This research presents the Adaptive Hybrid Agent (AHA) — a unified architecture that combines symbolic rule reasoning, large language model (LLM) inference, and continuous feedback-driven adaptation. The AHA architecture features a meta-controller that dynamically orchestrates between symbolic, neural, and hybrid reasoning paths according to context complexity, confidence thresholds, and feedback gradients derived from user interaction and system metrics. By employing multi-modal feedback including rule validation outcomes, engagement signals, and hallucination scores, the framework achieves self-calibrating behavior that improves both accuracy and adaptability under domain drift. Empirical evaluation across CRM workflows such as campaign generation, text-to-SQL analytic reasoning, and policy-aware message composition demonstrates significant improvements in factual grounding (+57%), task success rate (+14%), and engagement uplift (+25%) relative to static or single-modality baselines. These findings suggest that adaptive hybrid architectures enable reliable, auditable, and continually self-improving CRM agents, capable of aligning generative reasoning with enterprise governance.
