# AI Podcast Index

A statically generated directory of 43 AI personalities indexed by podcast appearances, research papers, open-source projects, and AI models. Each profile is enriched at build time with live data from GitHub and Hugging Face, and optionally with deep research generated by a LangGraph pipeline backed by DeepSeek.

## Stack

- **Frontend** — Next.js 16 (Turbopack), React 19, Tailwind CSS
- **Data** — Static JSON (Spotify episodes, arXiv papers, research profiles)
- **Enrichment** — LangChain LCEL pipeline fetching GitHub repos + Hugging Face models at build time
- **Deep Research** — LangGraph + DeepSeek pipeline (Python) generating structured bios, timelines, quotes, and contributions
- **Deployment** — Vercel (static generation, ISR 1h for person pages)

## Person Page Sections

Each `/person/[slug]` page renders (conditionally, based on available data):

| Section | Data Source |
|---------|------------|
| Profile hero (avatar, name, role, org) | `personalities.ts` |
| Open Source Projects (repos with stars, forks, languages, topics) | GitHub API (build-time) |
| AI Models (downloads, likes, pipeline tags) | Hugging Face API (build-time) |
| Tech Stack (language bar) | GitHub API (build-time) |
| Podcast Appearances (Spotify episodes) | `spotify_episodes.json` |
| Research Papers (arXiv links) | `personalities.ts` papers array |
| Deep Research (bio, timeline, contributions, quotes, social, sources) | `research/{slug}.json` |

## Research Pipeline

Generate a deep research profile for a personality:

```bash
export $(grep DEEPSEEK_API_KEY ~/.env)
python3 langgraph/cli.py research --slug athos-georgiou
```

The pipeline flow:

```
START → generate_queries → [search_web ×N] → check_urls → fetch_github → synthesize → export → END
```

- **generate_queries** — DeepSeek generates 8-12 targeted search queries
- **search_web** — DuckDuckGo search (12 results/query, fanned out in parallel)
- **check_urls** — Fetches full page content from top 12 URLs (skips social/news sites)
- **fetch_github** — GitHub profile + top repos via API
- **synthesize** — DeepSeek synthesizes all data into structured JSON profile
- **export** — Writes to `src/lib/research/{slug}.json`

### Evals

```bash
# Deterministic evals on existing JSON files (no API calls)
python3 -m src.graphs.person_research.evals --verbose

# Full pipeline + deepeval GEval metrics (costs API credits)
python3 -m src.graphs.person_research.evals --full
```

Deterministic evals: identity anchors, forbidden domains, structure depth, arXiv paper counts, URL validity, bio/name checks.

GEval metrics (DeepSeek as judge): biography quality, research completeness, source attribution.

## arXiv Pipeline

Fetch papers for a personality:

```bash
python3 langgraph/cli.py arxiv "Georgiou, A" --name "Athos Georgiou" --slug athos-georgiou
```
