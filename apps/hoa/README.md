# AI Podcast Index

A statically generated directory of 43 AI personalities indexed by podcast appearances, research papers, open-source projects, and AI models. Each profile is enriched at build time with live data from GitHub and Hugging Face, and optionally with deep research generated by a fully offline LangGraph pipeline powered by local MLX inference (Qwen2.5 on Apple Silicon).

## Stack

- **Frontend** — Next.js 16 (Turbopack), React 19, PandaCSS
- **Data** — Static JSON (Spotify episodes, arXiv papers, research profiles)
- **Enrichment** — LangChain LCEL pipeline fetching GitHub repos + Hugging Face models at build time
- **Deep Research** — LangGraph + MLX local inference (Python) generating structured bios, timelines, quotes, and contributions — fully offline, no API keys
- **Deployment** — Vercel (static generation, ISR 1h for person pages)


## Person Page Sections

Each `/person/[slug]` page renders (conditionally, based on available data):

| Section | Data Source |
|---------|------------|
| Profile hero (avatar, name, role, org) | `personalities.ts` |
| Open Source Projects (repos with stars, forks, languages, topics) | GitHub API (build-time) |
| AI Models (downloads, likes, pipeline tags) | Hugging Face API (build-time) |
| Tech Stack (language bar) | GitHub API (build-time) |
| Podcast Appearances (Spotify episodes) | `spotify_episodes.json` |
| Research Papers (arXiv links) | `personalities.ts` papers array |
| Deep Research (bio, timeline, contributions, quotes, social, sources) | `research/{slug}.json` |

## Research Pipeline

### Setup (one-time)

```bash
# Download the local model (~4.5 GB for 7B, ~2 GB for 3B)
python3 backend/download_model.py
# Or use the smaller model:
python3 backend/download_model.py --model mlx-community/Qwen2.5-3B-Instruct-4bit
```

### Run

```bash
python3 backend/research_pipeline.py --slug harrison-chase

# Use a specific model:
python3 backend/research_pipeline.py --slug harrison-chase --model mlx-community/Qwen2.5-3B-Instruct-4bit

# Or set via env var:
export MLX_MODEL=mlx-community/Qwen2.5-3B-Instruct-4bit
python3 backend/research_pipeline.py --slug harrison-chase
```

The pipeline runs 20 specialized agents in 3 phases:

- **Phase 1** — Intelligence Gathering: web search, GitHub, arXiv, HuggingFace, podcasts, news, videos
- **Phase 2** — Deep Analysis: biography, timeline, contributions, quotes, social, topics, competitive landscape, etc.
- **Phase 3** — Synthesis: quality evaluation, executive summary, interview question generation
- **Export** — Writes to `src/lib/research/{slug}.json`

All LLM inference runs locally via Apple MLX (Qwen2.5-Instruct). No API keys needed.

### Evals

```bash
cd backend
python3 -m pytest tests/ -v
```

Deterministic evals: identity anchors, forbidden domains, structure depth, arXiv paper counts, URL validity, bio/name checks.

GEval metrics (local MLX model as judge): biography quality, research completeness, source attribution.

## Environment
