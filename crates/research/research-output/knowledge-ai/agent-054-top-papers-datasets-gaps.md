## **2. Datasets Catalog: Key Educational Datasets**

### **2.1 Knowledge Tracing & Student Modeling Datasets**

**ASSISTments Datasets:**
- **ASSISTments 2009-2010**: 4,217 students, 325,637 responses, 110 skills
- **ASSISTments 2012-2013**: 19,840 students, 2,713,101 responses, 265 skills
- **ASSISTments 2015**: 19,917 students, 708,631 responses, 100 skills
- **ASSISTments Challenge 2017**: 1,709 students, 942,816 responses

**EdNet (2020):** [Large-Scale Hierarchical Dataset](https://doi.org/10.1007/978-3-030-52240-7_13)
- **Scale**: 784,309 students, 131,441,538 interactions
- **Hierarchy**: Course → Lecture → Question hierarchy
- **Features**: Timestamps, response times, hints, explanations
- **Applications**: KT, recommendation, student modeling

**Junyi Academy Dataset:**
- **Scale**: Millions of student interactions from Taiwanese platform
- **Features**: Math exercises with concept tagging
- **Temporal**: Longitudinal data spanning multiple years

### **2.2 MOOC & Online Learning Datasets**

**Open University Learning Analytics Dataset (OULAD):**
- **Students**: 32,593 students across 7 courses
- **Features**: Demographic data, assessment scores, clickstream data
- **Temporal**: Complete student journeys from registration to completion

**KDD Cup 2010 Educational Data Mining Challenge:**
- **Algebra 2008-2009**: 3,310 students, 809,694 transactions
- **Bridge to Algebra 2008-2009**: 6,043 students, 1,817,028 transactions

**MoocRadar (2023):** [Fine-grained and Multi-aspect Knowledge Repository](http://arxiv.org/abs/2304.02205)
- **Features**: Complete exercising context, knowledge components
- **Applications**: Cognitive student modeling in MOOCs

### **2.3 Automated Essay Scoring Datasets**

**Automated Student Assessment Prize (ASAP) Dataset:**
- **Essays**: 12,976 essays across 8 prompts
- **Scoring**: Holistic scores (1-6 or 1-3 scales)
- **Applications**: AES model training and evaluation

**TOEFL11 Corpus:**
- **Essays**: 12,100 essays from 11 language backgrounds
- **Applications**: Cross-lingual essay scoring

**DREsS Dataset (2024):** [Dataset for Rubric-based Essay Scoring on EFL Writing](https://doi.org/10.18653/v1/2025.acl-long.659)
- **Scale**: 48.9K samples with rubric-based scoring
- **Features**: Multi-trait assessment aligned with rubrics

### **2.4 Programming Education Datasets**

**Code.org Dataset:**
- **Scale**: Millions of student programming attempts
- **Features**: Block-based programming, solution attempts, debugging patterns

**GitHub Education Dataset:**
- **Repository Data**: Student projects, commit histories, code quality metrics
- **Applications**: Automated code assessment, plagiarism detection

**Programming Exercise Datasets:**
- Various datasets from online judges (Codeforces, LeetCode, HackerRank)
- Features: Problem statements, solutions, execution times, memory usage

### **2.5 Multimodal Learning Datasets**

**Classroom Simulacra Dataset (2025):** [Building Contextual Student Generative Agents](https://doi.org/10.1145/3706598.3713773)
- **Features**: Granularly annotated course materials, student interactions
- **Applications**: Student behavior simulation, virtual classroom modeling

**Affective Computing in Education Datasets:**
- **RAVDESS**: Speech emotion recognition for educational applications
- **SEANCE**: Text sentiment analysis for student writing
- **EEG-based cognitive load datasets**: Physiological signals during learning
